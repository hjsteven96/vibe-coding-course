# 🎯 Chapter 0: "바이브"를 위한 준비물

코딩을 시작하기 전에, 딱 4가지만 준비해 주세요.

## 1. Cursor AI 설치

[Cursor 공식 웹사이트](https://cursor.sh/)에서 다운로드해 설치합니다.

Cursor는 AI가 내장된 코드 에디터로, 여러분의 코딩 비서 역할을 합니다.

## 2. Node.js 설치

[Node.js 공식 웹사이트](https://nodejs.org/)에서 'LTS' 버전을 받아 설치합니다.

> Node.js는 Cursor AI가 터미널을 사용하기 위해 필요합니다.

## 3. 계정 준비

### GitHub 계정
- 코드를 저장할 '창고'입니다.
- [GitHub](https://github.com/)에서 무료로 가입할 수 있습니다.

### Vercel 계정
- 웹사이트를 세상에 공개할 '서버'입니다.
- [Vercel](https://vercel.com/)에서 GitHub 계정으로 로그인하면 됩니다.

## 4. Markdown Preview Enhanced 설치

이 튜토리얼 문서들을 예쁘게 보면서 따라하기 위해 마크다운 미리보기 확장 프로그램을 설치합니다.

### 설치 방법

1. Cursor AI를 엽니다.
2. 왼쪽 사이드바에서 **확장 프로그램(Extensions)** 아이콘을 클릭합니다. (또는 `Ctrl + Shift + X` / `Cmd + Shift + X` 단축키 사용)
3. 검색창에 `Markdown Preview Enhanced`를 입력합니다.
4. 검색 결과에서 **Markdown Preview Enhanced** (제작자: Yiyi Wang)를 찾아 `Install` 버튼을 클릭합니다.

### 사용 방법

1. Cursor에서 마크다운 파일(`.md`)을 엽니다.
2. 파일을 열고 우측 상단의 **미리보기 아이콘**(돋보기 모양)을 클릭하거나, `Ctrl + K` 후 `V`를 누릅니다. (macOS: `Cmd + K` 후 `V`)
3. 화면이 좌우로 나뉘면서 왼쪽에는 마크다운 코드, 오른쪽에는 예쁘게 렌더링된 문서가 보입니다!

> **팁:** 미리보기는 실시간으로 업데이트되므로, 이 튜토리얼을 따라하면서 계속 열어두면 편리합니다.

## ✅ 준비 완료!
