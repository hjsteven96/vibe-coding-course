# 🚀 My Vibe Canvas: 1시간 만에 AI로 내 웹사이트 런칭하기

> 코드를 한 줄도 몰라도 괜찮습니다.

우리는 코드를 '짜는' 게 아니라, AI에게 '지시'하고 '요구'하는 방법을 경험할 겁니다. **여러분의 'Vibe'가 곧 코드입니다.**

Cursor AI를 여러분의 '코딩 전문 비서'라고 생각하고, 지금부터 딱 1시간만 이 가이드를 따라와 보세요.

## 📚 튜토리얼 목차

이 튜토리얼의 목표는 **'내가 원하는 대로'** 기능이 붙는 웹사이트를 **'내 손으로'** 띄우고 **'친구에게 링크를 공유'**하는 경험입니다.

- 로컬에서 웹사이트 실행
- 첫 번째 디자인 적용

### [Chapter 2: "첫 번째 바이브" (Feature) - 원하는 기능 말하기](./chapter-2-첫번째-바이브.md)

- localStorage 활용
- 로컬 데이터 영구 저장

### [Chapter 5: "세상에 공개" (Deployment) - 전 세계에 자랑하기](./chapter-5-세상에-공개.md)

- 실제 도메인 획득

### [Chapter 6: "메모장 업그레이드" (Upgrade) - 진짜 메모 앱처럼 만들기](./chapter-6-메모장-업그레이드.md)

### [Chapter 7: "업데이트 배포" (Update Deployment) - 변경사항 세상에 공개하기](./chapter-7-업데이트-배포.md)

## 🎯 학습 목표

이 튜토리얼을 마치면:
- ✅ 코드 없이 AI로 웹사이트를 만들 수 있습니다
- ✅ localStorage와 클라우드 데이터베이스의 차이를 이해합니다
- ✅ 사이드바와 상세보기 같은 실용적인 UI를 구현할 수 있습니다

- ✅ 로그인/회원가입 기능을 구현할 수 있습니다
- ✅ 실제 도메인으로 서비스를 배포하고 공유할 수 있습니다
- ✅ 이미지 업로드와 자동 저장 같은 고급 기능을 추가할 수 있습니다
- ✅ Google Gemini API를 연동해서 AI 기능을 구현할 수 있습니다 (무료!)
- ✅ "내가 원하는 대로" AI에게 지시하는 방법을 익힙니다
