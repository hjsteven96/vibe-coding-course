# ✨ Chapter 2: "첫 번째 바이브" (Feature) - 원하는 기능 말하기

> **목표:** 내가 원하는 버튼을 AI에게 말해서 웹에 추가해 봅니다.

## Step 2.1: 'Vibe 추가' 버튼 만들기

1. `app/page.tsx` 파일이 열려 있는지 확인하세요.

2. 우측 **채팅창**에 `@app/page.tsx`를 입력해서 파일을 선택하고, 이렇게 지시합니다:

   > "@app/page.tsx 아까 만든 제목 밑에 'Vibe 추가'라는 이름의 버튼을 하나 만들어 줘. 이 버튼을 누르면 'Hello Vibe!'라는 경고창(alert)이 뜨게 해줘."

3. AI가 코드를 수정해줄 거예요. `Apply` 버튼을 누릅니다.

4. 브라우저에서 'Vibe 추가' 버튼을 눌러보고, 'Hello Vibe!'라는 경고창이 제대로 뜨는지 확인하세요.

---

### 오류가 발생했다면?

- 코드가 저장되지 않거나 에러 메시지가 뜬다면, 걱정하지 마세요.  
- 주로 다음과 같은 방법으로 해결할 수 있습니다:

  1. **터미널이나 웹 화면에 빨간 에러가 보이면, 모두 복사해서 AI에게 "이 에러 해결해줘!"라고 물어보세요.**  
  2. **브라우저 새로고침(F5), 서버 재시작(터미널에서 `Ctrl + C` 후 다시 `npm run dev` 입력)을 해보세요.**
  3. **그래도 안 된다면, 방금 코드를 지우고 '안전했던 단계'의 코드로 되돌려보세요.**

---

## Step 2.2: 기능 업그레이드 (입력창 + 버튼)

1. 경고창만으론 아쉬우니까, 다시 우측 **채팅창**을 엽니다.

2. `@app/page.tsx`를 입력하고, AI에게 더 구체적으로 요청해봅니다:

   > "@app/page.tsx 경고창은 그만. 'Vibe 추가' 버튼 위에 텍스트 입력창(input)을 하나 만들자. 그리고 'Vibe 추가' 버튼을 누르면, 내가 입력창에 쓴 글자가 경고창으로 뜨게 해줘."

3. AI가 코드를 수정할 거예요. (어떤 코드를 쓰더라도 걱정하지 마세요. 우리는 결과만 보면 됩니다.)

4. 브라우저에서 입력창에 글을 쓰고 'Vibe 추가' 버튼을 눌러 경고창이 입력한 대로 뜨는지 확인하세요.
