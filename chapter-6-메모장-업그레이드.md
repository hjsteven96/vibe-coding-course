# ✨ Chapter 6: "메모장 업그레이드" (Upgrade) - 진짜 메모 앱처럼 만들기

> **목표:** 메모장을 더 실용적으로 만듭니다. 사이드바에 메모 목록을 보여주고, 클릭하면 전체 내용을 볼 수 있게 합니다.

## Step 6.1: 제목과 본문 분리하기

지금은 메모가 한 줄만 입력되는 구조입니다. 진짜 메모 앱처럼 **제목**과 **본문**을 따로 입력받도록 바꿔봅시다.

1. 우측 **채팅창**에 `@app/page.tsx`를 입력하고:

   > "@app/page.tsx 메모에 제목과 본문을 따로 입력받게 바꿔줘. 제목 입력창과 본문 입력창(textarea)을 추가해 줘. 메모 추가 버튼을 누르면 제목과 본문이 함께 저장되게 해줘."

2. AI가 입력 UI를 두 개로 분리해 줄 겁니다.

3. 브라우저에서 테스트해보세요. 제목과 본문을 각각 입력하고 추가할 수 있나요?

## Step 6.2: 사이드바 레이아웃 만들기

이제 화면을 왼쪽(메모 목록)과 오른쪽(메모 내용)으로 나누는 레이아웃을 만들어 봅시다.

1. 우측 **채팅창**에:

   > "@app/page.tsx 화면을 2개로 나눠줘. 왼쪽에는 사이드바로 메모 목록이 보이게 하고, 오른쪽에는 선택한 메모의 전체 내용이 보이게 해줘. 
   > 
   > 사이드바의 각 메모에는 제목과 본문의 첫 줄 정도만 미리보기로 보여줘. 메모를 클릭하면 오른쪽에 전체 내용이 나타나게 해줘."

2. AI가 레이아웃을 2단으로 나누고, 클릭 기능을 구현해 줄 겁니다.

3. 브라우저에서 테스트해보세요:
   - 메모 여러 개를 추가합니다.
   - 왼쪽 사이드바에서 메모를 클릭하면 오른쪽에 전체 내용이 보이나요?

## Step 6.3: 입력 폼 위치 조정하기

새 메모를 추가하는 입력창은 어디에 있는 게 좋을까요? 상단이나 사이드바 위쪽이 편할 것 같습니다.

1. 우측 **채팅창**에:

   > "@app/page.tsx 새 메모 추가 폼(제목, 본문 입력창과 추가 버튼)을 화면 상단이나 사이드바 위쪽에 배치해 줘. 더 사용하기 편하게."

2. AI가 UI를 재배치해 줄 겁니다.

3. 본인이 원하는 위치가 아니라면, "좀 더 왼쪽으로", "상단에 고정해줘" 같은 피드백을 주세요!

## Step 6.4: 삭제 버튼 추가하기

선택한 메모를 삭제할 수 있는 버튼도 추가합시다.

1. 우측 **채팅창**에:

   > "@app/page.tsx 오른쪽 메모 상세 화면에 '삭제' 버튼을 추가해 줘. 삭제 버튼을 누르면 확인 메시지를 띄우고, 확인하면 그 메모를 삭제하게 해줘."

2. AI가 삭제 기능을 추가해 줄 겁니다.

3. 메모를 선택하고 삭제 버튼을 눌러보세요. 제대로 삭제되나요?

## Step 6.5: 편집 기능 만들기

메모를 수정할 수 있는 편집 기능을 추가해 봅시다.

1. 우측 **채팅창**에:

   > "@app/page.tsx 선택한 메모를 편집할 수 있는 기능을 추가해 줘. 오른쪽 메모 상세 화면에 '편집' 버튼을 추가하고, 편집 버튼을 누르면 제목과 본문을 수정할 수 있는 입력창으로 바뀌게 해줘. '저장' 버튼을 누르면 수정된 내용이 저장되고, '취소' 버튼을 누르면 원래 내용으로 돌아가게 해줘."

2. AI가 편집 기능을 구현해 줄 겁니다.

3. 브라우저에서 테스트해보세요:
   - 메모를 선택하고 편집 버튼을 클릭합니다.
   - 제목과 본문을 수정하고 저장 버튼을 누릅니다.
   - 변경사항이 제대로 저장되나요?
   - 취소 버튼을 누르면 수정 전으로 돌아가나요?

## 💡 테스트해보기

새로고침을 해도 잘 작동하나요? localStorage가 제대로 작동하고 있다면:
- ✅ 메모 추가 → 새로고침 → 메모가 남아있음
- ✅ 메모 삭제 → 새로고침 → 삭제한 메모가 없음
- ✅ 메모 선택 → 새로고침 → 선택이 해제됨 (정상)

## 🔥 6단계 성과

단순한 메모장이 진짜 메모 앱처럼 업그레이드 되었습니다! 이제 제목과 본문이 있고, 목록을 보며 선택할 수 있는 실용적인 앱이 되었네요.
